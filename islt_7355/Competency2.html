Competency 2 - Front-End Analysis and System Evaluations

Description
The ISLT 9466 Learning Analytics class taught me how to collect, analyze, and reflect upon the data in one of my classes that uses the Moodle learning management system. The artifact is a collection of four assignments where I used analytics to identify potentially ask students and suspect learning objectives. I located, cleaned, analyzed, and visualized the data borne through the .csv files. The files capture the student mouse-clicks when they open a file. Each learning objective connects with a video summary. I was able to track the number of activations for each video associated with each student, each learning objective, and each quiz and homework question.

Reflection
The class project had a direct and immediate impact on my teaching. I was able to identify at-risk student and also at-risk learning objectives. I learned about how (big) data is collected, the formats of the data, the cleaning and tidying of the data, and the visualizations of the data in a manner that tells a story into a communication dashboard. I learned about visualizations in Tableau and in Excel through using the PivotChart.
